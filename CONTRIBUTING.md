# Contributing to Agent Think Tank

Thanks for your interest in contributing! This is a personal productivity system that evolved from individual use to open source. All improvements are welcome.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/amazing-improvement`)
3. **Commit your changes** (`git commit -m 'Add amazing improvement'`)
4. **Push to the branch** (`git push origin feature/amazing-improvement`)
5. **Open a Pull Request**

## What We're Looking For

- **New agent variations** - Different personas that work with the same pattern
- **Implementation examples** - Code showing how to run this with various LLM APIs
- **Documentation improvements** - Clarifications, diagrams, tutorials
- **Use case stories** - How you're using the Think Tank in your life

## Guidelines

- Keep agent personalities distinct and consistent
- Maintain the strict output formats (makes synthesis possible)
- The Cook should always find tensions, not just summarize
- One action beats ten suggestions

## Questions?

Open an issue or start a discussion. This is as much a conversation as a codebase.
